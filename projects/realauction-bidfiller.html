RealAuction Auto Bidder

Chrome extension that streamlines tax lien auction bidding by automating premium calculations across hundreds of properties

Overview

RealAuction Auto Bidder is a Chrome extension designed to streamline the tax lien auction bidding process on the Colorado Tax Sale platform (El Paso County). The extension automates the tedious task of manually calculating and entering bid premiums for multiple properties, allowing investors to participate more efficiently in tax lien auctions.

The Problem

Tax lien auctions often feature hundreds of properties listed across multiple pages. Bidders compete by offering to pay a premium percentage above the face value of the tax lien. Manually calculating these premiums for dozens or hundreds of properties is:

  • Time-consuming: Each property requires individual calculation and entry
  • Error-prone: Mental math at auction speeds leads to mistakes
  • Inefficient: Investors lose competitive advantage due to slow data entry
  • Tedious: The repetitive nature makes it difficult to focus on investment strategy

The Solution

This Chrome extension provides a simple popup interface that:

  1. Accepts a premium percentage from the user (e.g., "15%" of face value)
  2. Automatically calculates the bid amount for every property on the current page
  3. Fills in all bid fields instantly with the calculated values
  4. Supports rounding to the nearest integer (optional)

Technical Implementation

Architecture

The extension follows Chrome's Manifest V3 architecture with three main components:

manifest.json - Permissions & Configuration

Defines extension metadata with minimal, security-focused permissions:

  • activeTab - Only operates on the currently active tab
  • scripting - Injects the bid-filling function into the page
  • storage - Reserved for saving user preferences
  • Host permissions restricted to https://elpaso.coloradotaxsale.com/*

popup.html - User Interface

Clean, minimal popup interface with:

  • Input field for percentage (e.g., "15")
  • Checkbox to enable/disable rounding
  • "Fill Bids" button to execute the automation

popup.js - Orchestration Logic

Coordinates the bid-filling process using Chrome's scripting API to inject functionality directly into the auction page.

Core Algorithm

The fillBids function performs precise calculations:

  1. Selects all premium input fields using the pattern input[id^="userBidEntry"]
  2. For each property:
    • Extracts the itemId from the input element
    • Finds the corresponding face value element #faceValue{itemId}
    • Parses the face value (removing currency symbols and formatting)
    • Calculates: bidValue = faceValue × (percentage / 100)
    • Optionally rounds up using Math.ceil()
    • Fills the input field with the calculated value
premiumInputs.forEach(input => {
  const itemId = input.getAttribute('itemid');
  const faceValueElement = document.querySelector(`#faceValue${itemId}`);
  
  if (faceValueElement) {
    let faceValueText = faceValueElement.getAttribute('price') 
                        || faceValueElement.textContent;
    let faceValue = parseFloat(faceValueText.replace(/[^0-9\.]/g, ''));
    
    if (!isNaN(faceValue)) {
      let calculatedBidValue = (faceValue * (bidValue / 100));
      if (roundUp) {
        calculatedBidValue = Math.ceil(calculatedBidValue);
      }
      input.value = calculatedBidValue.toFixed(0);
    }
  }
});

Error Handling

The extension includes robust validation:

  • Checks for valid bid percentage input (isNaN(bidValue) || bidValue <= 0)
  • Verifies face value elements exist before attempting to parse
  • Logs detailed console messages for debugging
  • Gracefully handles missing or malformed data

Installation & Usage

Installation

  1. Clone the repository: git clone https://github.com/rmfalco89/realauctionBidFiller.git
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" (toggle in top-right corner)
  4. Click "Load unpacked"
  5. Select the realauctionBidFiller directory

Usage

  1. Navigate to an active auction page on elpaso.coloradotaxsale.com
  2. Click the extension icon in the Chrome toolbar
  3. Enter your desired premium percentage (e.g., "15")
  4. Optionally enable "Round up to nearest integer"
  5. Click "Fill Bids"
  6. All bid fields will populate instantly
  7. Review and manually submit your selected bids

Design Philosophy

The extension follows a conservative automation approach:

  • Semi-automated rather than fully automated
  • User retains control over final bid submission
  • Transparent calculation with console logging
  • No persistent state or data collection

Why No Auto-Submit?

Giving users manual control over submission prevents accidental bids and maintains compliance with auction rules. The tool empowers users rather than removing their agency.

Technical Challenges

Dynamic DOM Manipulation

The auction site likely uses dynamic content loading. The extension handles this by:

  • Using attribute selectors (id^="userBidEntry")
  • Checking for element existence before manipulation
  • Providing detailed error logging

Data Parsing

Face values may appear in various formats ($1,234.56, 1234.56, etc.). The extension uses regex to strip non-numeric characters:

faceValueText.replace(/[^0-9\.]/g, '')
